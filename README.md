# OMDB API Telegram Bot

Telegram-бот для поиска информации о фильмах с использованием внешнего API.

## Возможности
- Поиск фильмов по названию
- Получение информации: рейтинг, описание, год выпуска
- Обработка пользовательских запросов
- Обработка ошибок (если фильм не найден)

## Задача проекта
Реализовать Telegram-бота с интеграцией внешнего API для получения и отображения данных в реальном времени.

## Технологии
- Python
- python-telegram-bot
- OMDb API
- httpx
- PostgreSQL
- Docker

## Архитектура и инфраструктура
- Используется Docker для контейнеризации приложения
- PostgreSQL в качестве основной базы данных
- Возможность быстрого развёртывания проекта через docker-compose

## Как это работает
1. Пользователь вводит название фильма
2. Бот отправляет запрос к OMDb API
3. Получает данные о фильме
4. Форматирует и отправляет ответ пользователю

## Технические особенности
- Интеграция с внешним API
- Работа с HTTP-запросами
- Обработка ошибок API
- Форматирование ответа пользователю
